Неопределенный интеграл

Неопределенным интегралом называют комплекс всех первообразных функции f(х).

Формулу для расчета неопределенного интеграла можно записать в такой форме:

\( \int f(x)dx = F(x) + C \)
  • f(x) - подынтегральная функция;
  • F(x) - первообразная функция функции f(x);
  • dx - дифференциал;
  • C - численная константа интегрирования.

ПРИМЕР Найти интеграл: \( \int (x^5 + \dfrac{1}{\sqrt{x}})dx \)
Решение: Интеграл суммы можно разложить на сумму интегралов: \( \int (x^5)dx + \int(\dfrac{1}{\sqrt{x}})dx \) Таким образом, каждый из интегралов преобразован в табличный вид. Решение можно найти с помощью таблицы: \( \dfrac{x^6}{6} + 2\sqrt{x} + C \)
Ответ: \( \dfrac{x^6}{6} + 2\sqrt{x} + C \)
